Optimizing Google Webfont

This demo shows how to optimize Google Web Fonts by subsetting the font set, specifying only certain font styles and using only certain font characters.

Google Webfont Specification

<link href='http://fonts.googleapis.com/css?family=Roboto+Condensed&text=log' rel='stylesheet' type='text/css'>

Serving Certain Characters

logo
If you take a closer look below, you will notice that the font family of Roboto Condensed only appplie to the character c
abcdefg
